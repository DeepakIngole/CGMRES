Continuation/GMRES method in C++

# Introduction
This project provides the implementation of the continuation/GMRES(C/GMRES) method, a fase computation algorithm of NMPC proposed by T. Ohtsuka, in C++.

# Requirement
- Cmake
- Eigen 3
